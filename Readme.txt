1. Internet Explorer에 맞춰서 했습니다
2. 중간에 SSL 통신이 끼어들면 문제가 생길 수 있으니 멈추거나 에러가 뜨면 다시 켜서 시도해주세요
3. 한 번 실행 후 KEYBOARD INTERRUPT로 종료하지 않을 경우나 EXCEPTION이 제대로 떠서 종료되지 않은 경우

4. 몇 개의 EXCEPTION을 handling 못해서 그런지(ssl 통신이거나 GET CONNECT HEAD 등으로 연결하지 않는 것들이 문제인 듯) 안 되는 경우가 가끔 있을 수 있으니
그 경우에도 다시 켜서 시도해주세요.
